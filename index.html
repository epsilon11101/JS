DESTRUCTURING

Is a useful JS feature to extract properties from objects and bind them to variables.
Example:

const hero = {
name: 'Batman',
realName: 'Bruce Wayne'
};
const { name, realName } = hero;
name; // => 'Batman',
realName; // => 'Bruce Wayne'

PROTOTYPES

All JS objects inherits properties and methods from a prototype
Example:
function Person(first,age) {
this.name= first;
this.age = age;
}
Person.prototype.presentation = function () {
console.log(`Hi im ${this.name} and im ${this.age} years old`);
};

PROTOTYPICAL INHERITANCE

Is the way to access to other prototypes properties, override its methods and create new methods
Example
function Dog(name,age,size) {
this.super = Animal
this.super(name, age);
this.size = size;
}
//DOG INHERITS FROM ANIMAL
Dog.prototype = new Animal();
Dog.prototype.constructor = Dog;

GETTERS AND SETTERS AND STATIC METHODS

An static method is a way to execute a method without instantiating a class
Get and Set is the way to access to attributes of the class and modify it
Example:
static hi() { console.log('Howdy there!'); }
get breed() {
return this.kind_breed; }
set breed(breed) {
this.kind_breed = breed; }
